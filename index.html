Densha Installer

Windows ve macOS kullanicilari icin hazir kurulum scriptleri. Scripti indirip calistirdiginizda densha-oto ve densha-ui repolari clone edilir, bagimliliklar kurulur ve Playwright Chromium yuklenir. Kurulum sonrasinda ayri launcher scriptleri ile iki proje birlikte acilabilir.

Windows
install-densha.cmd
Cift tiklayarak calistirin. Chromium dahil tum bagimliliklari kurar.
macOS
install-densha.sh
Bash veya zsh ile tek komutluk kurulum.
Windows
start-densha.cmd
Cift tiklayarak calistirin. Iki projeyi ayri PowerShell pencerelerinde baslatir.
macOS
start-densha.sh
Iki projeyi ayri Terminal pencerelerinde baslatir.
Dokumantasyon
README.md
Kullanim ve dagitim notlari.

Gerekenler

  • Node.js 20 veya ustu: Node.js indir
  • Git: Git indir
  • macOS icin Xcode Command Line Tools: Apple developer tools

Hizli Kullanim

  1. Node.js ve Git'i kur.
  2. Isletim sistemine uygun installer scriptini indir.
  3. Scripti lokal bilgisayarda calistir.
  4. Kurulum bitince `start-densha.cmd` (Windows) veya `start-densha.sh` (macOS) ile projeleri ac.
  5. Calisma akisi: `densha-ui` icin `npm run start`, `densha-oto` icin `npm run server`.
Scriptler eksik bagimliliklari kontrol eder ve hata verirse once yukaridaki resmi kurulum baglantilarini kullanin.
