Ilk adim: isletim sistemini sec.

Windows Kurulum

Kolay kurulum
  • Node.js ve Git yoksa asagidaki iki dosyayi indirip normal sekilde kur (Next, Next, Finish).

1) Gerekli Programlar

Node.js indir (Windows .msi) Kurup pencereyi kapat, bu sayfaya geri don. Git indir (Windows .exe) Kurup pencereyi kapat, bu sayfaya geri don.

2) Densha Dosyalari

install-densha.cmd Kurulumu yapar start-densha.cmd Sonraki acilislarda kullan
  1. Bu iki dosyayi AYNI klasore indir.
  2. install-densha.cmd dosyasini cift tikla ve kurulumun bitmesini bekle.
  3. Kurulum bittikten sonra uygulamayi her acmak istediginde start-densha.cmd dosyasini cift tikla.

macOS Kurulum

Kolay kurulum
  • Node.js, Git veya Xcode Command Line Tools yoksa once asagidaki butonlardan kur.

1) Gerekli Programlar

Node.js indir (macOS .pkg) Kurup pencereyi kapat, bu sayfaya geri don. Git indir (macOS installer) Kurup pencereyi kapat, bu sayfaya geri don. Xcode Command Line Tools Apple sayfasindan kur.

2) Densha Dosyalari

install-densha.sh Kurulumu yapar start-densha.sh Sonraki acilislarda kullan
  1. Bu iki dosyayi AYNI klasore indir.
  2. Terminal acip ayni klasorde chmod +x install-densha.sh start-densha.sh calistir.
  3. ./install-densha.sh ile kurulumu tamamla.
  4. Kurulum bittikten sonra uygulamayi her acmak istediginde ./start-densha.sh calistir.
